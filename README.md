# DryRun

**See algorithms think.**

DryRun is an interactive algorithm visualization platform that transforms code execution into a visual learning experience.

Traditional coding platforms tell you whether your solution works.

DryRun helps you understand why.

By combining execution tracing, dynamic visualizations, and algorithm-focused explanations, DryRun turns complex problem-solving processes into something learners can see, follow, and understand.

---

## Overview

Algorithms are often taught through static diagrams, code snippets, and debugger outputs.

The challenge is that most learners struggle to build a mental model of what is actually happening during execution.

DryRun bridges that gap by visualizing the internal state of algorithms as they run.

Arrays shift.

Pointers move.

Trees traverse.

Graphs expand.

Data structures evolve in real time.

The result is a learning experience focused on intuition rather than memorization.

---

## Core Principles

### Visual First

Concepts should be seen, not imagined.

### Learn the Reasoning

Understanding *why* a decision is made is more valuable than watching a line execute.

### Pattern-Based Thinking

Strong problem solvers recognize patterns, not solutions.

### Interactive Exploration

Learning happens through experimentation, observation, and feedback.
